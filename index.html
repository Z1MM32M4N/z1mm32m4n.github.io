TeXDown - www.texdown.org

For efficient Markdown note-taking infused with the power of LaTeX.

This is my favorite work-in-progress app. Targeting the technical CS/Math student, it allows people to efficiently combine the quickness and ease of Markdown note-taking with LaTeX for typesetting. gitorial - www.gitorial.com

With gitorial, making tutorials is as easy as commiting a repository to GitHub.

The product of TartanHacks 2014, this app won $750 worth of AWS credit and nearly placed overall. It reduces the process of creating a programming/code tutorial to three simple steps: make a GitHub repository where each commit is a step of a tutorial, sign in to gitorial to add some explanation text to each slide, and finally view your live tutorial.

This project never could have happened without the help of Aaron Gutierrez, Shyam Raghavan, and Westin Lohne. In the future, we have tons of amazing features planned, the most prominent of which being using git diffs to show changes between steps, so people can more easily follow the sometimes subtle changes between steps of a tutorial.
