15-251 Games!

TAs in 15-251 really like to give us board game proofs. I myself find that it's useful to be able to play these games to get a feel for their dynamics. Here are a few that I've taken the time to whip up into web games!

Also, if you wrote your own player, or if you added a feature to one of those already listed here, let me know! I'd be more than happy to include it.

Galumphing Banderwoozles

Pressing a tile turns it red and toggles the state of all it's neighbors. Try to see which ones you can solve by turning the entire board red!

Play!

Game of Thrones: Varys

This is the game that Varys proposes to Littlefinger. Can you get it to the state where all are blue but the middle dot is red?

Play!

A Song of Orbits

A simulator for the song of orbits activity: choose some initial values and step it out! See if you can find a pattern to the madness.

Play!

Magenta Wedding

Not a game, but useful to generate boards with random numbers and the winners highlighted.

Play!
